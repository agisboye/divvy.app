Divvy Privacy Policy

1. Background

1.1 This privacy policy applies to the Divvy application ("the Application"), which is published by Totus Labs ApS ("the Publisher").

1.2 By using the Application, the User accepts to be covered by this privacy policy (hereinafter "the Privacy Policy") and gives consent for the Publisher to collect and use information as described below.

1.3 The data controller for your personal information is the Publisher.

2. Collection and Storage of Information

2.1 The Publisher may collect the following personal information about the User: phone number, email, name, and profile picture. The Publisher may also collect information about the User's device, such as the device's IP address, operating system, and browser type.

2.2 Additionally, user-generated content such as expenses, messages, and comments are collected.

2.3 The Publisher uses a Google service to collect information about the use of the Application for statistical purposes. This includes information such as when the user has started the Application, how long the user spends in the Application, and which features the user utilizes. The information does not contain personal data and is used toimprove the Application.

2.4 The collected information is stored on servers located in the EU, which are controlled by the Publisher. The Publisher has entered into a data processing agreement with its subcontractor.

2.5 The information is stored as long as the User is registered in the Application. The User can at any time request to have their information deleted. See the section below.

3. Use of Information

3.1 The Publisher uses the information to make the Application available to the User.

4. Sharing of Information

4.1 The information collected by the Publisher is not shared with third parties.

5. Security and Protection

5.1 The Publisher has taken appropriate technical and organizational measures to prevent the User's personal information from being accidentally or unlawfully deleted, degraded, or lost, as well as to prevent unauthorized third parties from gaining knowledge of it or otherwise misusing it.

6. Right of Access

6.1 The User has certain rights regarding their information. These are outlined in the following points and can be exercised by contacting the Publisher through the Application or by email: hi@divvy.dk.

6.2 The User has the right to be informed about which personal information the Publisher has collected about them, as well as to receive a copy of this information.

6.3 The User has the right to have incorrect information corrected.

6.4 The User can withdraw their consent for the Publisher to collect information about the User and have this information deleted. If the User does this, the Publisher cannot make the Application available to the User.
